## Image Optimization Tips

1. **Resize** to max 1200px wide (phones don't need larger)
2. **Compress**: Use tinypng.com or squoosh.app
3. **Format**: JPEG for photos, WebP for better compression
4. **LQIP**: Generate tiny 20px-wide base64 placeholders at https://blurha.sh

Update `src/data/memories.json` with the actual filenames and LQIP strings.
